## 1. 기능

### 1.1 핵심 기능
- 웹 페이지 크롤링 및 데이터 추출
- 다양한 웹사이트 지원 가능 (구직사이트, 뉴스, 쇼핑몰 등)
- 데이터 저장 및 내보내기 (CSV, JSON 형식)
- 자동화된 스크래핑 스케줄링 (구현 예정)

### 1.2 부가 기능
- 멀티스레딩 지원
- 에러 핸들링 및 로깅
- 프록시 서버 지원
- 데이터 필터링 및 정제
